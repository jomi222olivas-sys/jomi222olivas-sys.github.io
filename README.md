# Jomi — Dev Portfolio

Portafolio personal construido con HTML, CSS y JavaScript vanilla. Sin frameworks, sin dependencias — puro código.

🌐 **Live:** https://jomi222olivas-sys.github.io

---

## Stack

- HTML5 / CSS3 / JavaScript (vanilla)
- Canvas API para partículas animadas
- Google Fonts: Space Grotesk + JetBrains Mono
- IntersectionObserver para scroll reveal
- GitHub Pages para el deploy

## Secciones

- **Hero** — presentación con efecto de tipeo
- **Sobre mí** — bio y stats
- **Skills** — lenguajes y tecnologías
- **Proyectos** — repos reales de GitHub
- **Servicios** — qué ofrezco
- **Contacto** — links
